## ğŸš€ Getting Started

### Prerequisites

- Node.js & npm (for Live Server)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm start
   ```
   
   The app will open automatically at `http://localhost:8080`

### Alternative: Open Directly

If you prefer not to use npm, simply open `index.html` in your browser or use VS Code's Live Server extension.
